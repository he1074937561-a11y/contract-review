## 1. 测试策略总览

### 1.1 测试金字塔

```
            ╱╲
           ╱  ╲          E2E 测试
          ╱    ╲         (Playwright)        — 少量
         ╱──────╲
        ╱        ╲      集成测试
       ╱          ╲     (FastAPI TestClient)  — 适量
      ╱────────────╲
     ╱              ╲   单元测试
    ╱                ╲  (pytest)             — 大量
   ╱──────────────────╲
```

| 层级 | 工具 | 目标覆盖率 | 运行频率 |
|------|------|-----------|---------|
| 单元测试 | pytest | > 80% 核心逻辑 | 每次提交 |
| 集成测试 | httpx.AsyncClient | > 70% API 端点 | 每次提交 |
| AI 审查测试 | pytest + 模拟 | 100% 审查流程 | 每次提交 |
| E2E 测试 | Playwright（P2） | 核心路径 | 发布前 |

### 1.2 测试范围矩阵

| 模块 | 单元测试 | 集成测试 | 说明 |
|------|---------|---------|------|
| 认证模块 | ✅ | ✅ | 注册、登录、Token 校验 |
| 合同管理 | ✅ | ✅ | 上传、列表、详情、删除 |
| 风险审查 | ✅ | ✅ | **审查引擎为核心，需重点覆盖** |
| 报告模块 | ✅ | ✅ | 生成、查询、导出 |
| AI 对话 | ✅ | ✅ | 对话、上下文管理 |
| 管理后台 | — | ✅ | 用户管理、系统配置 |
| 工具函数 | ✅ | — | JWT、密码哈希、配置 |

### 1.3 技术选型

| 工具 | 用途 | 选型理由 |
|------|------|---------|
| pytest | 测试框架 | Python 生态标准，异步支持好 |
| pytest-asyncio | 异步测试 | FastAPI/SQLAlchemy 均使用 async |
| httpx | HTTP 客户端 | FastAPI TestClient 基于 httpx |
| pytest-cov | 覆盖率 | 原生集成，报告丰富 |
| pytest-mock | Mock 工具 | 用于模拟 DeepSeek API 调用 |
| unittest.mock | Mock 标准库 | Python 内置，成熟稳定 |

    assert verify_password(password, hashed)

def test_hash_password_salt_unique():
    """相同密码两次哈希结果不同（盐值机制）"""
    password = "test_password"
    h1 = hash_password(password)
    h2 = hash_password(password)
    assert h1 != h2

def test_create_access_token_contains_sub():
    """Token 中包含用户 ID"""
    token = create_access_token({"sub": 1})
    payload = decode_access_token(token)
    assert payload["sub"] == "1"

def test_create_access_token_expiry():
    """Token 有过期时间"""
    token = create_access_token({"sub": 1})
    payload = decode_access_token(token)
    assert "exp" in payload

def test_expired_token_returns_none():
    """过期 Token 返回 None"""
    from datetime import timedelta
    to_encode = {"sub": 1, "exp": datetime.now(timezone.utc) - timedelta(hours=1)}
    token = jwt.encode(to_encode, settings.SECRET_KEY, algorithm="HS256")
    payload = decode_access_token(token)
    assert payload is None

def test_invalid_token_returns_none():
    """无效 Token 返回 None"""
    payload = decode_access_token("invalid.token.here")
    assert payload is None
